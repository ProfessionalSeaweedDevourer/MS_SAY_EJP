vim-tiny 삭제
	sudo apt-get remove vim-tiny

vim 설치
	sudo apt-get install vim

실행
	vi [파일명]

모드
	입력
		명령모드에서 
			a : 커서 뒤
			A : 행 끝
			i : 커서
			I : 행 처음
			s : 커서 한 글자 지우고
			S : 커서 행 지우고
			o : 다음 행에
			O : 한 행 뒤로 밀어내고
	명령
		기본모드
			입력 모드에서 esc
			실행 모드에서 esc

		커서 이동   
			h : 왼쪽
			l : 오른쪽
			k : 위쪽
			j : 아래쪽

			^ or 0 : 행 맨 앞으로
			$ : 행 맨 뒤로
			- : 이전 행 처음으로
			+ or Enter: 다음 행 처음으로
			
			H : 화면 맨 위로
			L : 화면 맨 밑으로
			Ctrl+b : 한 화면 위로
			Ctrl+f : 한 화면 아래로
			G : 맨 끝으로
			
			w : 다음 단어 첫 글자 위치로
			e : 다음 단어 마지막 글자 위치로

		문자열 관련
			x or del : 커서 위치 한 글자 삭제
			dw : 커서 위치 단어 잘라내기
			dd : 커서 위치 행 잘라내기

			yw : 커서 위치 단어 복사
			yy : 커서 위치 행 복사

			p : 커서 아래쪽 행에 붙이기					
			P : 커서 위쪽 행에 붙이기

		실행 취소
			Ctrl+z 아님 : fg로 돌아오기
		
			u : 명령 취소
			U : 해당 행에서 한 모든 명령 취소	
	실행
		/문자열 : 문자열 검색
		:%s/문자열1/문자열2/g : 문자열1->문자열2 바꾸기
		:set nu : 행 번호	
		:w : 저장
		:q : 종료
		:wq! : 저장 종료
