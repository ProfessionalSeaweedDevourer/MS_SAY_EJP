10. 계정 만들기
PC]
10-1. 이전 문법으로
	alter   session   set   "_ORACLE_SCRIPT"=true;
10-2. 계정 만들기
	create   user   아이디   identified   by   비번;
	create user kwon identified by 1;
10-3. role(권한세트) 부여
	grant   connect, resource   to   아이디;
	grant connect, resource to kwon;
10-4. tablespace 만들기
	create   tablespace    테이블스페이스명
	datafile   '2번폴더경로/파일명.dbf'
	size  500m;

	create tablespace kwonTS
	datafile 'C:/Kwon/odb/kwonTS.dbf'
	size 500m;
10-5. 계정에게 tablespace지정, 용량 제한 해제
	alter   user   아이디  
	default    tablespace   테이블스페이스명
	quota   unlimited   on   테이블스페이스명;

	alter user kwon default tablespace kwonTS quota unlimited on kwonTS;
-----
