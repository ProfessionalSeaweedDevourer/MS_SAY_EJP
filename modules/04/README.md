## 🇰🇷 한국어

이 모듈은 **교재·강의 기반 학습 중심**으로 진행되어, 본 저장소에는 코드 산출물이 포함되어 있지 않습니다. 학습 내용 자체는 AI-900 수료의 기반이 되었고, 뒤의 Module ⑥ (MS Azure AI) 실습을 지지하는 인프라 지식입니다.

### 🎯 학습 범위

- Azure Portal 탐색 · 구독·리소스 그룹·리전 개념
- Azure VM 프로비저닝 · SSH 접속 · Linux 기본 명령
- 네트워크 보안 그룹(NSG) · 공용 IP · 가상 네트워크 기초
- 스토리지 계정 · Blob/File 서비스 개요
- Azure 비용 구조 · 리소스 종료 라이프사이클

### 📎 산출물

이 모듈의 산출물은 소스코드가 아닌 **Azure Portal 조작 결과**(리소스 생성, VM 부팅 로그, 포트 설정 등) 이므로 저장소에 포함하지 않았습니다.

### 🔗 관련 증빙

본 모듈의 학습은 이후 Module ⑥ 의 Azure Cognitive Services 호출로 이어지며, 최종적으로 [../../assets/ai900-certificate.pdf](../../assets/ai900-certificate.pdf) (Microsoft Certified — Azure AI Fundamentals, 2026-02-12) 로 증빙됩니다.

## 🇺🇸 English

This module was taught primarily through **textbook and lecture-based material**, so no source artifacts live in this repository. The learning itself underpinned the AI-900 certification and supports the Azure Cognitive Services work in Module ⑥.

### 🎯 Scope of Study

- Azure Portal navigation · subscriptions · resource groups · regions
- Azure VM provisioning · SSH · Linux command-line basics
- Network Security Groups (NSG) · public IPs · virtual network fundamentals
- Storage accounts · Blob / File services overview
- Azure cost model · resource lifecycle and shutdown

### 📎 Deliverables

The deliverables for this module were **Azure Portal operations** (resource creation, VM boot logs, port configuration) rather than source code, so they are not part of this repository.

### 🔗 Evidence

Learning from this module flows into Module ⑥'s Azure Cognitive Services work, and is ultimately evidenced by [../../assets/ai900-certificate.pdf](../../assets/ai900-certificate.pdf) (Microsoft Certified — Azure AI Fundamentals, 2026-02-12).
