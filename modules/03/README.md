### 🎯 핵심 학습 주제

- 시맨틱 HTML → CSS 레이아웃 → 바닐라 JS → jQuery → ES6 → React 까지 **프론트엔드 진화 경로** 전 구간 체험

- The **full evolution of frontend**: semantic HTML → CSS layout → vanilla JS → jQuery → ES6 → React

- React components, props/state, hooks, routing fundamentals
